方法说明

手写 Inexact ALM 求解器

目标函数是 min ||L||_* + λ||S||_1, s.t. A = L + S。 把 NP-hard 的 rank/l0 松弛为核范数/l1，用交替方向法迭代求解。 只调用了 numpy.linalg.svd，其余全部手写。

低秩项更新 (SVT) L = SVT1/μ(A - S + Y/μ)
稀疏项更新 (Shrink) S = Shrinkλ/μ(A - L + Y/μ)
乘子更新 Y ← Y + μ(A - L - S)
惩罚系数递增 μ ← min(ρμ, μbar)
2 种底图 × 3 种破坏 = 6 个实验

纹理底图严格低秩（rank-4），人像底图做 rank-6 截断。不同底图 + 不同破坏类型，RPCA 的表现差异很大。
