## 一、起点：一个本以为很直觉的问题

这学期的作业二，题目问的是：**如何从一张被污染的图片里，还原出干净的原图？**

第一反应是用滤波——高斯模糊或中值滤波，把噪声"抹掉"。但很快发现问题：滤波不区分信号和噪声，细节和噪声一起消失了。更根本的困难是，在真实场景里，**我们根本不知道哪些像素被污染了**——没有污染掩码、没有先验信息，只有一张"坏掉的"图。

这让我意识到，直觉性的图像处理思路在这里行不通。问题需要被**重新建模**。

### 1.1 换一个框架：图像是矩阵

关键的转变发生在我重新审视"图像"这个概念的时候。一张 160×160 的灰度图，本质上是一个矩阵。如果图像内容规整——比如渐变纹理、人脸的主要轮廓——这个矩阵的**秩其实非常低**，用很少几个奇异向量就能近似描述全图信息。

反过来，文字遮挡、划痕、椒盐噪声只影响少数像素——从矩阵角度看，是**稀疏的异常项**。

这就给了我一个新的问题框架：把观测到的污染图 $A$ 分解为两部分：

$$A = L + S$$

- $L$：低秩矩阵，代表干净图像主体
- $S$：稀疏矩阵，代表噪声和遮挡

**只要这两个假设同时成立，L 和 S 就可以被数学上分离出来。**

### 1.2 凸优化与 Inexact ALM

直接最小化矩阵的秩和稀疏度是 NP-hard 问题。RPCA（鲁棒主成分分析）用了一个精妙的**凸松弛**：

$$\min_{L, S} \|L\|_* + \lambda \|S\|_1 \quad \text{s.t.} \quad A = L + S$$

核范数 $\|L\|_*$（所有奇异值之和）是秩的最紧凸下界；$\ell_1$ 范数 $\|S\|_1$ 是稀疏度的最紧凸下界。把 NP-hard 问题松弛成凸优化，代价几乎可以忽略不计——在满足特定条件时，两者的解完全等价。

我参考 Lin, Chen & Ma（2010）的论文，手写实现了 **Inexact ALM（非精确增广拉格朗日乘子法）**：交替更新 $L$（奇异值阈值操作 SVT）和 $S$（软阈值 Shrink），拉格朗日乘子 $Y$ 每轮加回残差，惩罚系数 $\mu$ 指数增长加速收敛。除了 `numpy.linalg.svd` 之外，其余全部手写。

### 1.3 实验结果：令人惊喜，也令人困惑

我设计了 **2 种底图 × 3 种破坏类型 = 6 个场景**的完整实验网格，覆盖文字遮挡、矩形缺块、椒盐噪声三种现实场景。

结果出乎意料地好：在 rank-4 合成纹理上，椒盐噪声几乎被**精确还原**（PSNR 趋向无穷，SSIM 接近 1.0）；人像的文字遮挡修复 PSNR 达到 29.9 dB。

但我也发现了算法的局限：**矩形缺块**的修复效果明显弱于文字遮挡，因为矩形缺块虽然稀疏，却过于"整齐"，导致 RPCA 对其稀疏性的判断出现混淆（support F1 很低）。

## 二、转折：洗干净的图，如何"贴进去"？

做完 RPCA 作业不久，作业四的题目来了：**Poisson 图像融合**。

我最初以为这是两个完全独立的问题——一个做修复，一个做融合。但当我真正动手实现 Poisson Editing 的时候，我突然看到了这两者的内在联系：

> RPCA 解决的问题是：**从混合信号中分离出我想要的部分**。  
> Poisson 解决的问题是：**把分离出来的部分，无缝地"贴"回另一张图里**。

两个问题，恰好是一条流水线的前后两段。

### 2.1 直接粘贴为什么失败

把 RPCA 还原的干净区域直接复制粘贴到另一张图上，边界会出现明显的色块跳变——人眼对不连续性极为敏感，哪怕 1 个像素的突变也会穿帮。

问题根源在于：直接复制约束的是**像素值的绝对大小**，而我们真正需要约束的是**变化的方式**——也就是梯度。

### 2.2 Poisson 方程：梯度决定融合走向

Poisson Image Editing 的核心思想是：让融合结果 $v$ 在区域 $\Omega$ 内部的**梯度**尽可能接近源图 $s$ 的梯度，同时在边界 $\partial\Omega$ 上等于目标图 $t$ 的值。这个问题自然地写成变分形式：

$$\min_v \int_{\Omega} |\nabla v - \nabla s|^2, \qquad v\big|_{\partial\Omega} = t\big|_{\partial\Omega}$$

对泛函取变分，Euler-Lagrange 方程恰好是经典的 **Poisson 方程**：

$$\Delta v = \Delta s \quad \text{in } \Omega, \qquad v = t \quad \text{on } \partial\Omega$$

这和稳态热传导完全同构：已知"内部热源"（源图的拉普拉斯）和"边界温度"（目标图边界值），求区域内的温度分布。

### 2.3 离散化与 SOR 实时求解

将图像离散化后，每个内部像素 $p$ 对应一个线性方程：

$$|N(p)| \cdot v_p - \sum_{q \in N(p) \cap \Omega} v_q = \sum_{q \in N(p)} (s_p - s_q) + \sum_{q \in N(p) \setminus \Omega} t_q$$

这等价于稀疏线性方程组 $A\mathbf{v} = \mathbf{b}$，矩阵 $A$ 对称正定，每行最多 5 个非零元素。

实现中我还解决了一个工程挑战：**如何在实时拖动时保证响应速度？** 方法是缓存遮罩拓扑结构（仅在遮罩变化时重建），拖动时只更新右端向量 $\mathbf{b}$，并用 **SOR（逐次超松弛，$\omega=1.5$）** 替代直接求解——拖动时 20–30 次迭代（约 10ms），松手后 100–200 次细化。

### 2.4 量化结果

四组测试样例，接缝跳变量降幅均在 **77%–87%**，平均处理时间约 48ms。

我还注意到一个有趣的对比：**Mixing Gradients**（在每条边上取源图和目标图中幅值更大的梯度）并不总是优于 Importing Gradients。当源对象本身结构清晰（如霓虹标志）时，保留其原始梯度更合适；当目标背景有强纹理时，Mixing 才有优势。**方法的选择取决于场景的语义**，而不仅仅是数值指标。

## 三、发现：两把工具，汇聚于医疗影像

做完两道作业，我开始思考一个更大的问题：**这两种数学工具在现实世界里用在哪里？**

查阅文献的过程中，我发现了一个让我非常兴奋的答案：这两种工具在**计算病理学（Computational Pathology）**领域几乎是天然的需求——而且恰好对应了今年的一篇重要论文 **nnMIL**（*A Generalizable Multiple Instance Learning Framework for Computational Pathology*，arXiv 2025）所面临的核心预处理挑战。

### 3.1 RPCA 在病理影像中的角色

全视野切片影像（Whole Slide Imaging，WSI）有一个典型的困扰：**染色的系统性偏差**。同一块组织，用不同批次的试剂、不同厂商的扫描仪处理，颜色会有明显差异——H&E 染色的蓝紫比例可能差 30% 以上。

这种染色偏差有一个重要的数学性质：它是**低秩的**。染色偏差是全局的、缓慢变化的系统性误差，而组织中真正有诊断价值的信息（细胞核形态、腺体排列、血管分布）才是"局部稀疏"的高秩特征。

换句话说，**A（原始病理切片）= L（染色背景结构）+ S（诊断性细节信息）**。RPCA 可以将两者分离：保留 $L$ 用于染色归一化，利用 $S$ 提取真正的组织信号。

此外，切片扫描过程中会出现气泡、折叠组织、刮痕等**局部伪影**，它们正是稀疏异常项 $S$，RPCA 可以自动检测并剥离。

在动态 PET 成像中，这一方法已有文献验证（*3D Tensor Based Nonlocal Low Rank Approximation in Dynamic PET Reconstruction*）：时间序列帧构成低秩矩阵，Poisson 计数噪声为稀疏项。

### 3.2 Poisson 融合在病理影像中的角色

WSI 的采集方式是**分块扫描后拼接**——扫描仪在高倍率下分块拍摄，再将所有 Patch 拼合成整张切片。这个过程中，相邻 Patch 的边界往往存在**亮度/颜色跳变**。

对于后续的机器学习模型（如 nnMIL），这些人工接缝是有害的噪声：模型可能把"接缝线"当作真实的组织边界特征来学习，产生虚假的诊断依据。

Poisson 图像融合正好可以解决这个问题：在相邻 Patch 的边界区域，用 Poisson 方程平滑过渡，消除亮度跳变，保持梯度连续性。**处理后的切片，接缝处不再是"人工产物"，模型看到的是物理上连续的组织图像**。

Mixing Gradients 模式在这里也有特殊价值：当相邻 Patch 边界两侧都有真实的组织纹理时，Mixing 模式会优先保留幅值更大的梯度，确保组织结构的连续性，而不是简单地让一侧"覆盖"另一侧。

### 3.3 nnMIL：下游任务

处理好的切片进入 **nnMIL** 框架。nnMIL 遵循多实例学习（Multiple Instance Learning，MIL）的范式：

- **Instance = Patch**：从预处理好的 WSI 上提取的图像块
- **Bag = Slide**：完整的一张切片
- **任务**：在仅有 Slide 级别标签（如"有无恶性肿瘤"）的情况下，预测切片的临床结论

nnMIL 的三个核心设计解决了传统 MIL 的瓶颈：

1. **标准化训练接口**：将变长 Bag 转为固定长度序列，支持大批次训练，降低对特征提取器选择的敏感性
2. **随机子空间注意力**：训练时随机采样特征子空间计算注意力权重，作为隐式正则化；推理时用完整特征空间
3. **子空间集成推理**：将特征空间均匀分割为多个子集，各子集预测平均后得到集成结果，同时自然给出不确定性估计

这里有一个有趣的逻辑：**RPCA 预处理的质量直接影响 nnMIL 注意力机制的准确性**。如果染色伪影没有被去除，模型的注意力会错误地落在与诊断无关的区域；如果 Patch 接缝没有被平滑，模型可能把接缝线当作"可疑区域"。两者的预处理质量，是 nnMIL 能否正常工作的前提。

### 3.4 三者的数学统一性

回望这条路，最让我着迷的是这三个方法在数学框架上的统一性：

| 方法 | 优化目标 | 先验假设 | 数学工具 |
|------|----------|----------|----------|
| RPCA | $\min \|L\|_* + \lambda\|S\|_1$ | 低秩 + 稀疏 | 凸优化 + ALM |
| Poisson 融合 | $\min \int_\Omega |\nabla v - \nabla s|^2$ | 梯度平滑 | 变分法 + PDE |
| nnMIL 注意力 | $\min \mathcal{L}_{cls} + \text{正则}$ | 判别性稀疏 | 随机化 + 集成 |

三者都是**变分优化框架**——先验不同（低秩先验 / 梯度平滑先验 / 判别先验），尺度不同（像素级 / Patch 级 / 切片级），但共享"最优化一个能量函数"的数学语言。

这也意味着，它们理论上可以被纳入一个**联合优化框架**：在病理影像的端到端学习中，RPCA 项约束特征的低秩性，Poisson 项约束特征的空间平滑性，分类损失项约束判别能力——三者联合正则化，共同优化一个目标。

## 四、结语

这两道作业，我本来以为是两个孤立的练习。但当我开始追问"这些工具真正有用在哪里"的时候，一条意想不到的路径展开了：

> 图像被噪声污染 → **RPCA 分离信号与异常**  
> → 分离出的信号需要无缝融合 → **Poisson 方程保持梯度连续**  
> → 融合后的干净 Patch 送入模型 → **nnMIL 从 Patch 集合中做切片级诊断**

每一步的数学，都在为下一步创造条件。这也许就是"数学建模"这门课最想告诉我们的事情：数学工具本身不是孤立的——它们在真实问题里彼此咬合，像齿轮一样协同转动。
